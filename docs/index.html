Turn talking-head videos into scroll-stopping shorts — automatically.

Upload a video in any language. AUTO VISUAL AI transcribes it, understands it, generates word-synced AI visuals, writes animated English captions, and renders a cinema-quality 9:16 cut — with a human-in-the-loop approval step.

What it does

A fully automated, 16-step pipeline — every stage runs as an idempotent background job.

01 Upload
02 Extract audio
03 Transcribe (Whisper)
04 Word timestamps
05 Translate → English
06 Topics & keywords
07 Visual ideas
08 Generate images/video
09 Animated captions
10 Build timeline
11 Approve
12 Render (Remotion)
13 Quality gate
14 Export 4K

Highlights

Engineered like a real product — not a demo.

🌐

Multilingual → English captions

Auto-detects any spoken language (Telugu, Hindi, Urdu, Arabic, Spanish…), keeps the original audio, and renders accurate English subtitles.

🔑

One key, local-first

Runs on a single API key (OpenAI or Replicate). Files are stored on local disk by default — no AWS required to run it.

✨

Word-level animated captions

Karaoke / TikTok / Hormozi styles with active-word highlighting, synced to Whisper word timings.

🎬

Remotion 9:16 compositor

AI visuals on top, creator facecam below, captions overlaid — composed in React and rendered to 1080p→4K.

✅

Human-in-the-loop approval

Review and approve / regenerate every generated asset before the final render is built.

🧩

Swappable AI providers

Clean adapters for Whisper, Claude/GPT, Flux/SDXL, Runway/Kling/Pika — pick any with one env var.

Run it yourself

Local-first — no cloud account needed.
git clone → pnpm install → pnpm docker:up (or local Postgres/Redis) → pnpm db:push → pnpm dev, then add a single API key in the Settings page and upload a video. Full instructions in the repo README.
